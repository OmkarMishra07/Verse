# Verse

Verse is a modern Android music player and real-time jamming application. It features a unique classic iPod-style click wheel navigation interface blended with a sleek, translucent glassmorphism design. 

With Verse, you can not only listen to your favorite local music, but also join "Jam Sessions" with your friends to sync music playback in real-time, all while chatting in the same app.

## Features

- **Classic Click Wheel Navigation:** Rotate to scroll through menus and long-press the center button to open quick access menus.
- **Jam Sessions:** Create or join real-time chat rooms with friends.
- **Synchronized Playback:** Play, pause, or change a song, and it instantly syncs across all devices in the room.
- **Modern UI/UX:** A stunning dark theme featuring translucent glassmorphic elements and smooth animations.
- **Media Controls:** Full foreground service support for music playback, including lock screen and notification controls.



## How to Run Locally

To build and run the application on your own machine, follow these steps:

**Prerequisites:**
- [Android Studio](https://developer.android.com/studio) installed on your computer.
- An Android device or emulator running Android 7.0 (API level 24) or higher.

**Steps:**
1. **Clone the repository**
   ```bash
   git clone https://github.com/OmkarMishra07/Verse.git
   ```
2. **Open the project in Android Studio**
   - Launch Android Studio.
   - Select **Open** and choose the `Verse` directory you just cloned.
3. **Sync Gradle**
   - Allow Android Studio to download the necessary dependencies and sync the project.
4. **Run the App**
   - Connect your physical Android device via USB/Wireless Debugging or start an Android Emulator.
   - Click the green **Run** button (or press `Shift + F10`) to build and deploy the app to your device.

## Technologies Used
- **Kotlin** & **Jetpack Compose** for the UI.
- **Firebase Realtime Database** for chat and jam session synchronization.
- **AndroidX Media3 (ExoPlayer)** for reliable music playback.
